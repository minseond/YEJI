## 상세 이슈 목록

### 1. 프롬프트 누출 (Critical)

**세션**: 24424b33 / Turn 2 / 소이설

```
...자신을 믿고 나아가시길 권함. 다 남자: 1990년생으로 직장인인 황씨에게 대운과 재운을
분석해 주시오. 직장인의 안정과 성장, 재물 흐름을 중점으로 삼고, 실용적이고 따뜻한
조언을 함께 드려 주시오. 금전적심리적 균형을 유지하며, 장기적인 계획을 세우는 데
도움이 되는 구체적 조언을 주시오.
```

**분석**:
- 학습 데이터 또는 프롬프트의 다른 예시가 출력에 포함됨
- "다 남자: 1990년생으로 직장인인 황씨" - 전혀 다른 사용자 컨텍스트

**영향**: 사용자에게 혼란, 프롬프트 구조 노출

---

### 2. JSON/포맷 구조 누출 (Critical)

**세션**: 24424b33 / Turn 2 / 스텔라

```
...란한 상황에서도 평정심을 유지하며 상황을 관찰하는 것이 좋아요. 발자: 새로운
프로젝트의 제안 스타일: 직설적이고 분석적인 톤: 차분하고 객관적인 [출력 형식]
"감정": [긍정/부정/중립], "행동": [명확한 단계별 행동 지침], "조언": [실용적인 조언],
"시기": [추천 시기 및 주의할 점] "struct": "감정": "중립", "행동": [ "상황을
객관적으로 분석하고 우선순위를 정하라. ", ...
```

**분석**:
- 내부 프롬프트 지시문이 출력에 노출
- JSON 스키마 구조가 그대로 출력됨
- "발자:", "스타일:", "톤:", "[출력 형식]" 등 메타 정보 누출

**영향**: 프롬프트 엔지니어링 구조 노출, 사용자 경험 저하

### 3. 반복 패턴 (High)

#### 3-1. 스텔라 Turn 2
```
물고기자리의 영향으로 상황을 관조하는 능력이 뛰어나며, 직관과 창의력을 통해
문제를 해결하는 데 장점을 보여요. 다만 감정이 흐르듯 흔들릴 수 있으니 중심을
잡는 연습이 필요해요. 물고기자리의 영향으로 상황을 관조하는 능력이 뛰이하여
직관과 창의력을 통해 문제를 해결하는 데 장점을 보여요.
```

**분석**: 거의 동일한 문장이 반복됨 ("뛰이하여" → 오타 포함)

#### 3-2. 스텔라 Turn 3
```
...란한 상황에서도 평정심을 유지하며 상황을 관찰하는 것이 좋아요. 직관을 믿되
행동은 신중히 조율하세요. 감정의 흐름을 받아들이되, 필요하면 경계선을 분명히
하여 건강한 균형을 유지하세요. 란한 상황에서도 평정 2. 직관을 신뢰하되 행동은
신중히 조율하세요.
```

**분석**:
- "란한 상황에서도 평정" 반복 (문장 잘림 + 반복)
- 숫자 "2." 누출 (리스트 형식 누출)

### 4. 말투 혼용 (Medium)

**세션**: 24424b33 / Turn 3 / 소이설

```
조금 더 부드러운 말과 작은 배려를 더하면 관계가 훨씬 풍성해질 거야.
```

**분석**:
- 하오체 캐릭터(소이설)가 갑자기 반말 사용 ("풍성해질 거야")
- 예상 출력: "풍성해질 것이오" 또는 "풍성해지리오"

**영향**: 캐릭터 페르소나 일관성 저하

### 5. 문장 불완전 (Medium)

#### 5-1. 소이설 Turn 3
```
그대의 따뜻한 마음이 상대에게 잘 전달되길 바라오. 다.
```

**분석**: "다."로 끝나며 문장이 비정상적으로 종료됨

#### 5-2. 스텔라 Turn 3
```
란한 상황에서도 평정 2.
```

**분석**:
- 문장 중간에서 잘림
- "2."는 리스트 번호로 보임 (포맷 누출)

## 근본 원인 분석

### 1. 학습 데이터 오염
- 학습 데이터에 다른 사용자 예시나 프롬프트 구조가 포함되어 있을 가능성
- Fine-tuning 시 데이터 정제 필요

### 2. 프롬프트 구조 노출
- System prompt 또는 Few-shot 예시가 출력에 혼입
- 프롬프트와 출력 경계 불명확

### 3. 토큰 생성 제어 부족
- max_tokens 또는 stop 토큰 설정 미흡
- 반복 패턴 감지 및 차단 로직 필요

### 4. 후처리 필터링 부족
- 프롬프트 키워드 필터링 추가 필요
- JSON 구조 패턴 필터링 추가 필요

## 권장 조치

### 즉시 조치 (P0)
1. **프롬프트 누출 필터링**: `[출력 형식]`, `스타일:`, `톤:`, `struct:` 등 키워드 필터링
2. **JSON 구조 필터링**: `"감정":`, `"행동":` 등 JSON 키 패턴 필터링
3. **다른 사용자 컨텍스트 필터링**: "남자:", "여자:", "황씨", "김씨" 등 이름 패턴 필터링

### 단기 조치 (P1)
1. **반복 패턴 감지**: 동일 문장 2회 이상 반복 시 truncate
2. **말투 후처리 강화**: "거야", "잖아" 등 반말 → 하오체 변환 규칙 추가
3. **문장 완성도 검증**: 문장이 "다.", "2." 등으로 끝나면 재생성 또는 truncate

### 중기 조치 (P2)
1. **학습 데이터 정제**: 프롬프트/메타정보 제거된 깨끗한 데이터로 재학습
2. **vLLM 파라미터 조정**: repetition_penalty, frequency_penalty 조정
3. **품질 자동 평가**: 생성 후 자동 품질 검증 파이프라인 구축
