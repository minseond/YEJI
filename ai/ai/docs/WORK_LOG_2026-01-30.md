| **에러** | `'FiveElements' object has no attribute 'get'` |
| **원인** | `tikitaka_service.py:251`에서 `.get('dominant')` 호출, Pydantic 모델에 없음 |

**수정 내용**:

```python
class FiveElements(BaseModel):
    @property
    def dominant(self) -> str:
        """우세 오행 레이블 반환"""
        max_elem = max(self.elements_list, key=lambda x: x.percent)
        return max_elem.label

    @property
    def weak(self) -> str:
        """약한 오행 레이블 반환"""
        min_elem = min(self.elements_list, key=lambda x: x.percent)
        return min_elem.label

    def get(self, key: str, default: str = "") -> str:
        """dict 호환용 get 메서드"""
        if key == "dominant":
            return self.dominant
        # ...
```
