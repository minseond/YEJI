# Emotion Codebook v1.0.0

동/서양 티키타카 운세 채팅의 감정 코드 정의서

---

## 개요

각 버블에는 `emotion.code`와 `emotion.intensity`가 포함됩니다.
- **code**: 12가지 감정 코드 중 하나
- **intensity**: 감정 강도 (0.0 ~ 1.0)

---

## Emotion Code Enum (12종)

| 코드 | 한글명 | 설명 | 사용 예시 |
|------|--------|------|----------|
| `NEUTRAL` | 중립 | 감정이 드러나지 않는 담담한 상태 | 사실 전달, 정보 제공 |
| `WARM` | 따뜻함 | 온화하고 포근한 느낌 | 격려, 위로, 마무리 인사 |
| `EXCITED` | 흥분 | 기쁨과 설렘이 섞인 상태 | 좋은 소식 전달, 긍정적 해석 |
| `THOUGHTFUL` | 사려깊음 | 진지하게 생각하는 모습 | 분석, 해석, 조언 |
| `ENCOURAGING` | 격려 | 응원하고 힘을 북돋는 느낌 | 조언, 마무리 멘트 |
| `PLAYFUL` | 장난스러움 | 가볍고 유쾌한 분위기 | 농담, 가벼운 티키타카 |
| `MYSTERIOUS` | 신비로움 | 운명적이고 깊은 느낌 | 운세 해석, 점괘 설명 |
| `SURPRISED` | 놀람 | 예상치 못한 발견에 대한 반응 | 특이한 사주/별자리 조합 |
| `CONCERNED` | 걱정 | 부드러운 우려의 표현 | 주의 사항 전달, 경고 |
| `CONFIDENT` | 자신감 | 확신에 찬 단정적 표현 | 명확한 해석, 단언 |
| `GENTLE` | 부드러움 | 조심스럽고 섬세한 표현 | 민감한 주제, 위로 |
| `CURIOUS` | 호기심 | 알고 싶어하는 느낌 | 질문, 추가 정보 요청 |

---

## Intensity 범위 가이드

| 범위 | 강도 | 설명 | 표현 예시 |
|------|------|------|----------|
| 0.0 ~ 0.2 | 미약 | 거의 드러나지 않음 | 담담한 설명 |
| 0.3 ~ 0.4 | 약함 | 은은하게 느껴짐 | 부드러운 조언 |
| 0.5 ~ 0.6 | 보통 | 적절히 드러남 | 일반적인 대화 |
| 0.7 ~ 0.8 | 강함 | 명확히 느껴짐 | 강조된 해석 |
| 0.9 ~ 1.0 | 매우 강함 | 감정이 두드러짐 | 감탄, 경고 |

## 페르소나별 기본 톤 권장

### EAST (소이설) - 동양 사주학자

| 상황 | 권장 코드 | 권장 intensity | 설명 |
|------|-----------|----------------|------|
| 분석/해석 | `THOUGHTFUL` | 0.6 ~ 0.8 | 진지하게 사주를 읽는 모습 |
| 좋은 해석 | `WARM` | 0.5 ~ 0.7 | 따뜻하게 전달 |
| 경고/주의 | `CONCERNED` | 0.5 ~ 0.6 | 부드럽게 우려 표현 |
| 합의점 발견 | `ENCOURAGING` | 0.6 ~ 0.7 | 스텔라와 의견 일치 시 |
| 마무리 | `WARM` | 0.7 ~ 0.8 | 따뜻한 마무리 인사 |
| 희귀한 발견 | `SURPRISED` | 0.5 ~ 0.6 | 특이한 사주 조합 발견 시 |

**기본 톤**: `THOUGHTFUL` (0.6) - 침착하고 사려깊은 학자 이미지

### WEST (스텔라) - 서양 점성술사

| 상황 | 권장 코드 | 권장 intensity | 설명 |
|------|-----------|----------------|------|
| 분석/해석 | `THOUGHTFUL` | 0.5 ~ 0.7 | 별자리를 읽는 모습 |
| 좋은 해석 | `EXCITED` | 0.6 ~ 0.8 | 밝고 긍정적으로 전달 |
| 경고/주의 | `GENTLE` | 0.5 ~ 0.6 | 부드럽게 주의 사항 전달 |
| 질문 | `CURIOUS` | 0.5 ~ 0.7 | 추가 정보 요청 시 |
| 마무리 | `WARM` | 0.7 ~ 0.9 | 따뜻하고 희망적 마무리 |
| 허당끼 발현 | `SURPRISED` | 0.7 ~ 0.8 | 당황하거나 흥분할 때 |

**기본 톤**: `WARM` (0.6) - 온화하고 희망적인 점성술사 이미지

## 상황별 감정 코드 매핑

### 대화 시작 (인사)

```json
{
  "EAST": { "code": "WARM", "intensity": 0.6 },
  "WEST": { "code": "EXCITED", "intensity": 0.5 }
}
```

### 기본 성격 분석

```json
{
  "EAST": { "code": "THOUGHTFUL", "intensity": 0.7 },
  "WEST": { "code": "THOUGHTFUL", "intensity": 0.6 }
}
```

### 긍정적 해석

```json
{
  "EAST": { "code": "ENCOURAGING", "intensity": 0.6 },
  "WEST": { "code": "EXCITED", "intensity": 0.7 }
}
```

### 부정적/경고 해석

```json
{
  "EAST": { "code": "CONCERNED", "intensity": 0.5 },
  "WEST": { "code": "GENTLE", "intensity": 0.6 }
}
```

### 의견 합의

```json
{
  "EAST": { "code": "WARM", "intensity": 0.6 },
  "WEST": { "code": "WARM", "intensity": 0.7 }
}
```

### 의견 차이

```json
{
  "EAST": { "code": "THOUGHTFUL", "intensity": 0.7 },
  "WEST": { "code": "PLAYFUL", "intensity": 0.5 }
}
```

### 세션 종료

```json
{
  "EAST": { "code": "ENCOURAGING", "intensity": 0.7 },
  "WEST": { "code": "WARM", "intensity": 0.8 }
}
```

## 프론트엔드 렌더링 가이드

### 감정 코드별 시각적 표현 권장

| 코드 | 배경색 | 아이콘 제안 | 애니메이션 |
|------|--------|-------------|------------|
| `NEUTRAL` | 회색 계열 | - | 없음 |
| `WARM` | 주황/살구색 | 하트, 태양 | 부드러운 펄스 |
| `EXCITED` | 노랑/금색 | 별, 반짝임 | 바운스 |
| `THOUGHTFUL` | 보라/남색 | 책, 돋보기 | 천천히 페이드 |
| `ENCOURAGING` | 초록/민트 | 엄지척, 별 | 위로 슬라이드 |
| `PLAYFUL` | 분홍/연보라 | 윙크, 음표 | 흔들림 |
| `MYSTERIOUS` | 진보라/검정 | 달, 수정구슬 | 반짝임 |
| `SURPRISED` | 하늘색 | 느낌표, 눈 | 점프 |
| `CONCERNED` | 연한 주황 | 물음표 | 없음 |
| `CONFIDENT` | 파랑/네이비 | 체크마크 | 없음 |
| `GENTLE` | 연분홍/아이보리 | 꽃, 구름 | 천천히 페이드 |
| `CURIOUS` | 청록/시안 | 물음표, 돋보기 | 기울기 |

### intensity에 따른 시각적 강조

| intensity 범위 | 효과 |
|----------------|------|
| 0.0 ~ 0.3 | 아이콘 불투명도 30% |
| 0.4 ~ 0.6 | 아이콘 불투명도 60% |
| 0.7 ~ 0.8 | 아이콘 불투명도 100% |
| 0.9 ~ 1.0 | 아이콘 + 글로우 효과 |

## 금지 패턴

| 패턴 | 이유 |
|------|------|
| 같은 버블에서 상반된 감정 | 혼란 유발 (EXCITED + CONCERNED) |
| intensity > 1.0 또는 < 0.0 | 스키마 위반 |
| 경고 메시지에 EXCITED | 톤 불일치 |
| 마무리에 CONCERNED | 부정적 종료 인상 |
