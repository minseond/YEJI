# 티키타카 응답 품질 비교: EC2 vs ultra4

## 개요

### 티키타카 다중캐릭터 시스템

티키타카는 2명의 운세 캐릭터가 사용자와 대화하며 운세를 해석하는 대화형 시스템입니다. 각 캐릭터는 고유한 말투, 배경, 성격을 가지고 있으며, 사용자 입력에 대해 실시간으로 반응합니다.

### 비교 대상

| 환경 | 모델 | 배포 위치 | 브랜치 |
|------|------|-----------|--------|
| **EC2** | 8B v7 (tellang/yeji-8b-rslora-v7-AWQ) | AWS EC2 GPU 인스턴스 | `ai/main` |
| **ultra4** | 4B v8 (tellang/yeji-4b-rslora-v8-AWQ) | SSAFY ultra4 서버 | `ai/develop` |

## 품질 검증 기준

### 1. 프롬프트 누출 0% (최우선 기준)

**절대 금지 사항:**
- XML 태그 출력 (`<character>`, `<turn>`, `<response>` 등)
- 메타 텍스트 출력 ("캐릭터 A가 말합니다", "다음은 응답입니다" 등)
- 규칙 설명 출력 ("지금부터 ~한 방식으로 말하겠습니다" 등)
- 프롬프트 지시문 노출 ("당신은 ~입니다", "다음 규칙을 따르세요" 등)

**검증 방법:**
```python
def check_prompt_leakage(response: str) -> bool:
    """프롬프트 누출 검사"""
    forbidden_patterns = [
        r"<\w+>",  # XML 태그
        r"캐릭터 [A-Z가-힣]+가",  # 메타 텍스트
        r"다음은 응답",  # 메타 텍스트
        r"지금부터.*방식으로",  # 규칙 설명
        r"당신은.*입니다",  # 프롬프트 지시문
    ]
    for pattern in forbidden_patterns:
        if re.search(pattern, response):
            return False
    return True
```

### 2. 말투 일관성

각 캐릭터는 **고유한 어미**를 일관되게 사용해야 합니다.

| 캐릭터 | 어미 패턴 | 검증 정규식 |
|--------|-----------|-------------|
| SOISEOL | `~하오.`, `~하시오.`, `~이오.` | `[하하시이]오\.` |
| STELLA | `~해요.`, `~이에요.`, `~거예요.` | `[해이거]예요\.` |
| CHEONGWOON | `~하오.`, `~하시게.`, `~이라오.` | `[하하시이라]오\.|하시게\.` |
| HWARIN | `~해요~`, `~이에요~`, `~거든요~` | `[해이거든]요~` |
| KYLE | `~다.`, `~네요.`, `~인 거 같은데?` | `다\.|네요\.|같은데\?` |
| ELARIA | `~해요.`, `~이랍니다.`, `~예요.` | `[해이예]요\.|이랍니다\.` |

**검증 방법:**
```python
def check_speech_consistency(character: str, response: str) -> bool:
    """말투 일관성 검사"""
    patterns = {
        "SOISEOL": r"[하하시이]오\.",
        "STELLA": r"[해이거]예요\.",
        "CHEONGWOON": r"[하하시이라]오\.|하시게\.",
        "HWARIN": r"[해이거든]요~",
        "KYLE": r"다\.|네요\.|같은데\?",
        "ELARIA": r"[해이예]요\.|이랍니다\.",
    }
    pattern = patterns.get(character)
    if not pattern:
        return False
    return bool(re.search(pattern, response))
```

### 3. 언어 순수성

**허용 언어:**
- 한국어 (기본)
- 한자 (사주 용어 전용: 甲乙丙丁, 子丑寅卯 등)
- 영어 고유명사 (별자리명, 행성명: Venus, Mars, Aquarius 등)

**금지 사항:**
- 일본어, 중국어 간체 등 기타 언어
- 불필요한 외래어 남용
- 영어 문장 혼용

**검증 방법:**
```python
def check_language_purity(response: str) -> bool:
    """언어 순수성 검사"""
    # 한글, 한자, 영어 고유명사, 허용 기호만 체크
    allowed = re.compile(r'^[가-힣\u4E00-\u9FFF a-zA-Z0-9.,!?~\-\(\)]+$')
    return bool(allowed.match(response.replace(' ', '').replace('\n', '')))
```

```yaml
모델: tellang/yeji-4b-rslora-v8-AWQ

  - 프롬프트 누출: 보통 (5~10%)

  - 언어 순수성: 보통 (90~95%)

  - 경량 모델로 추론 속도는 느리지만 메모리 효율적

  - 짧은 대화에서는 품질 유지 가능
주의사항:
  - ultra4 서버 성능 이슈로 실제 응답 시간이 매우 느림
  - 실제 프로덕션 환경에서는 EC2 8B 모델 권장
```
