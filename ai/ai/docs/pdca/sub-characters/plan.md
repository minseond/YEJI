# Plan: 서브 캐릭터 티키타카 품질 강화

## 목표
4개 서브 캐릭터(청운/화린/카일/엘라리아)의 티키타카 대화 품질을 90점 이상으로 달성

## 현재 구현 기준 (변경 불가)

| 캐릭터 | 말투 스타일 | 핵심 특징 |
|--------|------------|----------|
| 청운 | 하오체/하게체 | 시적, 비유, 현자의 위엄 |
| 화린 | 해요체 | 나른하면서 뼈있는, 비즈니스 |
| 카일 | 반말+존댓말 혼용 | 도박 비유, 쿨한 태도 |
| 엘라리아 | 해요체 | 우아한 공주, 왕실 품위 |

## 평가 기준 (90점 만점)

```yaml
필수_표현: 30점
  - 말투 어미 패턴: 10점
  - 캐릭터 특유 표현: 10점
  - 호칭/자칭 패턴: 10점

금지_표현: -10점/개
  - 합니다체 (청운/카일)
  - 하오체 (화린/카일/엘라리아)
  - 반말 (엘라리아)

톤_일관성: 30점
  - 캐릭터 톤 키워드 매칭률

자연스러움: 30점
  - 응답 길이 적절성
  - 반복 패턴 없음
```

## 검증 방법

1. **오프라인 테스트**: 샘플 응답 품질 평가
2. **E2E 테스트**: 실제 API 호출로 LLM 응답 평가
3. **반복 검증**: 90점 미달 시 Few-shot 조정 후 재테스트 (최대 5회)

## 예상 결과

- 4개 캐릭터 모두 90점 이상 달성
- 티키타카 3턴 연속 대화 자연스러움 확보
- Few-shot 패턴 문서화

## 리스크

1. LLM 응답 노이즈 (외래 문자, 반복)
2. 톤 키워드 매칭률 변동성
3. vLLM 서버 응답 지연

## 일정

- Phase 1: 품질 평가 프레임워크 완성 ✅
- Phase 2: E2E 테스트 실행 및 기준선 측정
- Phase 3: 미달 캐릭터 Few-shot 조정 (최대 5회 반복)
- Phase 4: 통합 검증 및 문서화
