## 근본 원인 분석

### 합니다체 문제 (핵심)
- vLLM 모델(tellang/yeji-8b-rslora-v7-AWQ)이 합니다체를 선호
- 프롬프트로 제어 어려움 (모델 학습 데이터 영향)
- 가능한 해결책:
  1. 모델 파인튜닝 (하오체/해요체 데이터셋)
  2. 후처리로 합니다체 → 하오체/해요체 변환 (규칙 기반)
  3. 다른 모델 사용

### 토큰 누출 문제
- `\buser\b` 정규식이 줄 끝에서 실패
- 해결: `\buser\b` → `user` (word boundary 제거)

### 한자 필터링 문제
- 문장 끝 한자 필터링 정규식 미작동
- 해결: 정규식 수정 필요
